What is expansion?
Refers to expansion of parameters.
Expansion, meaning the interpreter converts certain symbols into new text / tokens.

When?
Expansion occurs when certain symbols are put into certain context.
For instance, on its own, the symbol foo means nothing
However then foo is a variable with the string "hello" assigned to it, foo refers to a value.
When preceded by a $, bash will expand foo into "hello"
