An interactive shell reads commands from user input on a tty.

Allows a user to interact w/ the shell.

When a user opens a terminal, the OS provides a shell.

The user interacts with the OS through the shell.  This is an example of an interactive shell.

The shell through which a user interacts can be controlled by a user's profile.

Interactive shell vs interactive scripts
-------------------------------------------------
An interactive shell is a non-scripted cli in which a user is presented
with a prompt and can interact with the shell.

Interactive shells provide a prompt to the user.

An interactive script is one that requires user input for certain
functionality.

Commands like read are examples of interactive scripts.
